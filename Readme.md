# Java Pre-DSA Advanced problems

## Overview

This repository contains a set of 15 advanced Java programming problems designed to strengthen core problem-solving skills before moving into Data Structures and Algorithms (DSA).

The solutions focus on implementing logic from first principles without relying on built-in shortcuts or library methods.

---

## Objectives

* Develop strong problem-solving skills using Core Java
* Improve logical thinking and algorithm design
* Practice handling edge cases effectively
* Build a solid foundation for DSA and technical interviews

---

## Constraints

* No use of built-in shortcuts such as sorting utilities or hash-based collections
* Each problem is implemented using basic language constructs
* Every problem is written in a separate Java file
* Each file contains its own main method for independent execution

---

## Problem Categories

### Number Logic

* Reverse Integer with Overflow Handling
* Armstrong Number (Dynamic Digits)
* Strong Number

### Loops and Conditions

* Prime Numbers in Range with Count
* GCD and LCM without built-in methods
* Perfect Number Check

### Arrays

* Move All Zeros to End
* Find Duplicate Element without extra space
* Leaders in Array
* Left Rotate Array by K Steps
* Subarray with Given Sum

### Strings

* Longest Substring Without Repeating Characters
* String Compression

### Patterns

* Hollow Pyramid Pattern

### Mixed Logic

* Frequency of Elements without using map structures
